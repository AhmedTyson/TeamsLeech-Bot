# 📡 TeamsLeech Bot

> **🚧 Work in Progress** — This project is under active development. No functional code has been written yet.

TeamsLeech Bot is a personal automation tool that monitors Microsoft Teams lecture recordings across 6 university subjects and delivers new recordings directly to Telegram Saved Messages. It runs entirely on GitHub Actions free tier, uses OAuth2 refresh_token authentication with the Microsoft Graph API, and requires zero local machine involvement after initial setup. Triggered on-demand via Telegram commands, the bot dynamically discovers Teams drives, presents per-subject inline buttons, and streams selected recordings to Telegram with real-time progress updates — all at zero cost.
